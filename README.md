# MeuTCC
Meu primeiro repositório. Estou colocando aqui meu trabalho de conclusão de curso técnico em que fiz um sistema web / desktop interligado com banco de dados. O Objetivo principal eram o aprendizado, então, pode não ter ficado o melhor código do mundo mas, fácil de entender e aplicar.


Para abrir o projeto e rodar em sua máquina primeiramente é necessário que você tenha o XAMPP instalado. Ele será o servidor que servirá os arquivos php do meu projeto para exibir na tela. Usei o xampp em meu tcc e não tive problemas com ele. 

Depois de instalado, você precisará extrair os arquivos do site.rar para dentro da pasta "htdocs" que fica, geralmente, no diretorio "C:\xampp\htdocs"; 
Em seguida, será necessário abrir o painel de controle do xampp e startar o servidor (Apache e MySQL)
Logo, você precisará criar o banco de dados, executando, no seu navegador, o arquivo "criarBancoTeste.php" que estará dentro da pasta PHP, dentro da pasta do site que você extraiu.

Para executar esse arquivo, no seu navegador você digitará algo como http://localhost/site/php/criarBancoTeste.php
Depois de executado,  a pagina retornará algumas mensagens de sucesso dizendo que o banco foi criado com sucesso e que já foram inseridas duas pessoas na tabela pessoas, indicando uma delas um administrador do site, com acesso à página restrita, e um cliente, com acesso apenas a área do cliente. 

Depois de feito, é só acessar a página inicial do site, digitando em seu navegador http://localhost/site/
Aqui está o projeto.

Todas as páginas funcionam, com cadastro de documentos de clientes, alteração de dados e exclusão. Com exceção do página do carrinho que ficou por fazer. Fora o carrinho, as demais todas funcionam. 

Espero que gostem. O código não é o melhor dos melhores e nem o mais profissional, foi feito apenas para aprendizado como já citei.
